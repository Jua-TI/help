# help

Este repositorio tem como ideia disponibilizar uma arquitetura exercicios, resoluções, explicações/comentarios e codigos de apoio para auxiliar no aprendizados de novos integrantes no curso de Sistemas de Informação da UFPI. Tendo como foco inicial as duas primeiras materias de programação, Algoritmos 1 e 2.

Sera visto de maneira gradual o aprendizado, focando no que é dado de maneira padrao nas materias mas tambem visualizando outros assuntos por fora que ajudam a ter mais conhecimento util para linguagem C.

As listas não são de autoria nossa, são da FACOM (Faculdade de Computação) da Universidade Federal de Uberlândia.
