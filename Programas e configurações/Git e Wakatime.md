# Passos para instalação e configuração do Git e Wakatime

## O que é?

### Git

Git é um sistema de controle de versão distribuído que permite rastrear alterações no código fonte. Ele facilita o trabalho colaborativo, mantém histórico completo de modificações, permite criar branches para desenvolvimento paralelo e oferece a capacidade de reverter para versões anteriores. Git é essencial para desenvolvimento moderno e integra-se com plataformas como GitHub, GitLab e Bitbucket.

### Wakatime

Wakatime é uma ferramenta que rastreia automaticamente o tempo gasto codificando em diferentes linguagens de programação, projetos e arquivos. Ela se integra com IDEs e editores de código para coletar dados sobre produtividade. Wakatime fornece análises detalhadas sobre hábitos de codificação, linguagens mais utilizadas e tempo investido em cada projeto, ajudando desenvolvedores a entender e otimizar sua produtividade.
