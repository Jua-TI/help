# Como destacar uma palavra

- **Negrito**: `**palavra**` → **palavra**
- *Itálico*: `*palavra*` → *palavra*
- `Código`: `` `palavra` `` → `palavra`

## Como associar um diretório a uma palavra (link)

Use o formato:

`[texto do link](caminho/do/diretorio)`

Exemplos:

- [Programas e configurações](Material%20de%20apoio/Programas%20e%20configurações/)
- [Material de Apoio](Material%20de%20Apoio/)
