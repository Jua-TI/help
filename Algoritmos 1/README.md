## Como usar

1. Abra a pasta da atividade desejada.
2. Consulte o enunciado da atividade, se houver.
3. Registre suas solucoes em Suas Respostas/.

## Como compilar

1. Abra o terminal na pasta do arquivo .c.
2. Compile com:

- gcc arquivo.c -o programa

1. Execute:

- Windows: .\programa.exe
- Linux/macOS: ./programa

Opcional (varios arquivos .c):

- gcc *.c -o programa
