# Lista - Strings em C (Algoritmos 1)

## Formato padrao para todas as questoes

- **Entrada:** valores lidos do teclado.
- **Saida:** valores exibidos na tela.

## 1) Leitura e impressao

Leia uma palavra e imprima a mesma palavra.

- **Entrada:** uma palavra.
- **Saida:** a palavra lida.

## 2) Tamanho da string

Leia uma palavra e calcule o tamanho (quantidade de caracteres).

- **Entrada:** uma palavra.
- **Saida:** o tamanho da string.

## 3) Contagem de vogais

Leia uma frase e conte quantas vogais existem.

- **Entrada:** uma frase.
- **Saida:** a quantidade de vogais.

## 4) Contagem de consoantes

Leia uma frase e conte quantas consoantes existem.

- **Entrada:** uma frase.
- **Saida:** a quantidade de consoantes.

## 5) Substituir caractere

Leia uma frase e dois caracteres: antigo e novo. Substitua todas as ocorrencias do caractere antigo pelo novo.

- **Entrada:** uma frase, um caractere antigo e um caractere novo.
- **Saida:** a frase com as substituicoes.

## 6) Inverter string

Leia uma palavra e imprima a palavra invertida.

- **Entrada:** uma palavra.
- **Saida:** a palavra invertida.

## 7) Verificar palindromo

Leia uma palavra e verifique se e palindromo.

- **Entrada:** uma palavra.
- **Saida:** "Palindromo" ou "Nao palindromo".

## 8) Comparar duas strings

Leia duas palavras e verifique se sao iguais.

- **Entrada:** duas palavras.
- **Saida:** "Iguais" ou "Diferentes".

## 9) Concatenacao

Leia duas palavras e mostre a concatenacao delas.

- **Entrada:** duas palavras.
- **Saida:** a concatenacao.

## 10) Copia de string

Leia uma palavra, copie para outra string e imprima a copia.

- **Entrada:** uma palavra.
- **Saida:** a string copiada.

## 11) Primeira ocorrencia de caractere

Leia uma palavra e um caractere. Encontre a primeira posicao do caractere na palavra.

- **Entrada:** uma palavra e um caractere.
- **Saida:** a posicao ou -1 se nao existir.

## 12) Ultima ocorrencia de caractere

Leia uma palavra e um caractere. Encontre a ultima posicao do caractere na palavra.

- **Entrada:** uma palavra e um caractere.
- **Saida:** a posicao ou -1 se nao existir.

## 13) Remover espacos

Leia uma frase e remova todos os espacos.

- **Entrada:** uma frase.
- **Saida:** a frase sem espacos.

## 14) Contar palavras

Leia uma frase e conte quantas palavras existem.

- **Entrada:** uma frase.
- **Saida:** a quantidade de palavras.

## 15) Maior palavra da frase

Leia uma frase e mostre a maior palavra encontrada.

- **Entrada:** uma frase.
- **Saida:** a maior palavra.

## 16) Converter para maiusculas

Leia uma frase e converta todos os caracteres para maiusculas.

- **Entrada:** uma frase.
- **Saida:** a frase em maiusculas.

## 17) Converter para minusculas

Leia uma frase e converta todos os caracteres para minusculas.

- **Entrada:** uma frase.
- **Saida:** a frase em minusculas.

## 18) Ordenar caracteres

Leia uma palavra e ordene seus caracteres em ordem crescente.

- **Entrada:** uma palavra.
- **Saida:** a palavra com caracteres ordenados.

## 19) Verificar anagrama

Leia duas palavras (apenas letras) e verifique se sao anagramas.

- **Entrada:** duas palavras.
- **Saida:** "Anagrama" ou "Nao anagrama".

## 20) Analise completa

Leia uma frase e calcule: tamanho, quantidade de vogais, consoantes e digitos.

- **Entrada:** uma frase.
- **Saida:** tamanho, vogais, consoantes e digitos (um por linha).
